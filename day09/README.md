## Topic: Modify Operators (`++` and `--`)

Today I learned about the **increment** and **decrement** operators in C.  
These operators are used to quickly **increase or decrease** the value of a variable.

### **Pre-increment (`++x`)**
- Increases the value *before* the expression is evaluated.

### **Post-increment (`x++`)**
- Increases the value *after* the expression is evaluated.

- Decreases the value after use.
